#definition
### Types
- `G` : Set
- $*$ : $G \times G \to G$
### Definition
$(G, *)$ is called a *group*  if it satisfies the following axioms:
- Associativity: $\forall a, b, c \in G, (a * b) * c = a * (b * c)$.
- Identity Element: $\exists e, \forall a, e * a = a * e = a$.
- Inverse Element: $\forall g \in G, \exists h^{-1} \in G, g * g ^ {-1} = 1$.
